# AutoGPT: 텍스트 생성 및 완성을 위한 자율 AI 도구 AutoGPT는 LLM(Large Language Models)이 독립적으로 생각하고, 계획하고, 작업을 실행할 수 있게 해주는 자율 AI 도구입니다. 이 도구를 사용하면 AI가 합리적으로 생각하고 결정을 내릴 수 있으므로 더 이상 입력을 제공할 필요가 없습니다. 이 기사에서는 컴퓨터에 AutoGPT를 설치하고 사용하는 방법에 대해 설명합니다. ## Auto-GPT를 사용하기 전에 알아야 할 사항 AutoGPT를 사용하기 전에 알아야 할 몇 가지 사항은 다음과 같습니다. - AutoGPT를 설치하고 사용하기 위해 강력한 컴퓨터가 필요하지 않습니다. 괜찮은 컴퓨터라면 그 일을 할 것입니다. 무거운 작업은 클라우드의 OpenAI API에 의해 수행됩니다. - GPT-4 API에 액세스할 수 있는 경우 AutoGPT가 생각하고 결론을 내리는 데 훨씬 더 효과적이므로 가장 잘 작동합니다. 그러나 일반 OpenAI API를 GPT-3.5 모델과 함께 사용할 수도 있습니다. - Auto-GPT는 많은 토큰을 사용하므로 무료 신용 한도가 소진되면 청구 세부 정보를 추가하고 지출 한도를 설정해야 합니다. - 무료 버전의 ChatGPT와 달리 Auto-GPT는 인터넷에 연결하여 모든 주제에 대한 최신 정보를 찾을 수 있습니다. 따라서 이를 사용하여 모든 웹 페이지에 액세스하고 정보를 캡처할 수 있습니다. ## PC에서 Auto-GPT를 설정하는 방법 PC에 [AutoGPT](https://docs.kanaries.net/ko/articles/autogpt-4)를 설치하려면 다음 단계를 따르세요. ### 1단계: Python 및 Pip 설치 첫 번째 단계는 공식 웹 사이트에서 최신 버전의 Python을 다운로드하여 컴퓨터에 설치하는 것입니다. Python을 설치하기 전에 Windows에서 "Add python.exe to PATH"를 활성화해야 합니다. 설치 후 Windows 터미널을 열고 다음 명령을 실행하여 설치를 확인합니다. ``` 핍 --버전 ``` ### 2단계: Auto-GPT를 사용하기 위한 API 키 추가 Auto-GPT의 GitHub 릴리스 페이지로 이동하여 "소스 코드(zip)"를 클릭하여 ZIP 파일을 다운로드합니다. ZIP 파일을 추출하고 "Auto-GPT" 폴더를 복사합니다. 폴더를 원하는 위치에 붙여넣습니다. 메모장이나 텍스트 편집기를 사용하여 열어야 하는 ".env.template" 파일을 찾기 위해 폴더를 엽니다. OpenAI 웹 사이트로 이동하여 API 키를 받으십시오. 계정이 없으면 무료 계정을 만들고 "새 비밀 키 만들기"를 클릭하십시오. API 키를 복사합니다. 텍스트 편집기로 이동하여 “OPENAI_API_KEY” 섹션에 API 키를 붙여넣습니다. Pinecone.io를 열고 무료 계정을 만드세요. 왼쪽 사이드바에서 "API 키"를 클릭하고 오른쪽 창에서 "API 키 만들기"를 클릭합니다. "autogpt"와 같은 이름을 지정하고 "키 만들기"를 클릭합니다. 버튼을 클릭하여 "키 값"을 복사합니다. 텍스트 편집기를 다시 열고 PINECONE_API_KEY 옆에 붙여넣습니다. "환경" 아래의 값을 복사합니다. PINECONE_ENV 옆에 붙여넣습니다. ".env.template" 파일을 저장합니다. .env.template을 .env로 이름을 바꿉니다. ### 3단계: Windows, macOS 및 Linux에 Auto-GPT 설치 [Auto-GPT를 설치](https://docs.kanaries.net/ko/articles/auto-gpt-install)하려면 다음 단계를 따르세요. 1. API 키를 추가한 후 Auto-GPT를 실행할 차례입니다. "Auto-GPT" 폴더를 마우스 오른쪽 버튼으로 클릭하고 "경로로 복사"를 선택합니다. 2. 이제 터미널을 열고 cd를 입력하고 공백을 추가한 다음 위에서 복사한 경로를 붙여넣습니다. 터미널을 마우스 오른쪽 버튼으로 클릭하여 경로를 빠르게 붙여넣을 수 있습니다. 그런 다음 Enter 키를 누르면 Auto-GPT 폴더로 이동합니다. macOS 및 Linux 사용자는 폴더 자체 내에서 간단히 마우스 오른쪽 버튼을 클릭하고 터미널을 열 수 있습니다. 3. 여기에 있으면 아래 명령을 실행하십시오. Auto-GPT를 실행하는 데 필요한 모든 라이브러리를 자동으로 설치합니다. ``` pip 설치 -r requirements.txt ``` 4. 다음으로 아래 명령을 실행하면 시스템에서 Auto-GPT가 시작됩니다. ``` 파이썬 -m autogpt ``` ## PC에서 Auto-GPT를 사용하는 방법 Auto-GPT를 설치했으면 이제 사용법을 배울 차례입니다. 따라야 할 단계는 다음과 같습니다. 1. [Auto-GPT](https://docs.kanaries.net/ko/tutorials/ChatGPT/autogpt)는 초기 실행 시 AI 이름을 지정하라는 메시지를 표시합니다. 예를 들어 저는 Amazon에서 물건을 찾을 수 있는 Shopper-GPT라는 도구를 개발하고 있습니다. 특정 사용 사례에 대한 AI를 구축하지 않으려면 이 영역을 비워두고 Enter 키를 누릅니다. 기본적으로 Entrepreneur-GPT 이름을 로드합니다. 2. 다음 단계는 AI의 기능을 지정하는 것입니다. 3. 다음으로 자율 AI에 대한 일련의 목표를 설정합니다. 여기에서 당신이 성취하고자 하는 바를 AI에게 말할 수 있습니다. 데이터를 텍스트 또는 PDF 파일로 저장하도록 요청할 수 있습니다. 모든 정보를 얻은 후에 종료를 요청할 수도 있습니다. 4. 이제 Auto-GPT가 생각하기 시작합니다. 조치가 취해질 때 승인하라는 메시지가 표시됩니다. 확인하려면 "y"를 입력하고 Enter 키를 누릅니다. 웹 사이트에 연결하고 데이터를 수집할 수 있습니다. 5. AI의 생각, 정당성, 계획을 읽을 수 있습니다. 적절한 종류의 정보를 생성하기 위해 비판(일종의 부정적인 제안)도 제공합니다. 그런 다음 작업을 완료합니다. ## 자동 GPT 사용 Auto-GPT는 다양한 작업을 도와줄 수 있는 사용하기 쉬운 자율 AI 도구입니다. 다음은 몇 가지 예입니다. ### 텍스트 생성 텍스트 생성은 Auto-GPT의 주요 사용 사례 중 하나입니다. 이 도구는 다음을 포함하여 긴 자료를 만드는 데 사용할 수 있습니다. 제품 설명 및 완전한 에세이. Auto-GPT 인터페이스에서 "텍스트" 버튼을 클릭하고 텍스트를 생성하라는 메시지를 입력합니다. 예를 들어 요가의 이점에 대한 기사를 작성하려면 검색창에 "요가의 이점"을 입력하고 "생성"을 선택하십시오. 그런 다음 프롬프트에 따라 Auto-GPT가 단락과 제목이 있는 전체 기사를 생성합니다. 텍스트의 길이, 온도(출력 예측 불가능성을 제어함) 및 AI가 텍스트 생성을 시도해야 하는 횟수는 모두 수정될 수 있습니다. ### 텍스트 요약 Auto-GPT를 사용하여 텍스트를 빠르게 요약할 수도 있습니다. 이렇게 하려면 "요약" 버튼을 클릭하고 요약하려는 텍스트를 붙여넣습니다. Auto-GPT는 키워드와 가장 중요한 사항이 포함된 텍스트 요약을 생성합니다. ### 작업 완료 Auto-GPT를 사용하면 작업이 자동으로 완료될 수도 있습니다. 예를 들어 애플리케이션을 사용하여 소셜 미디어 게시를 계획하거나 이메일 응답을 자동화할 수 있습니다. 이를 수행하려면 Auto-GPT 인터페이스에서 "작업" 버튼을 클릭하여 자동화하려는 작업을 선택하십시오. 예를 들어 이메일 답변을 자동화하려면 "이메일"을 선택하고 화면의 지시를 따르십시오. - 여기에서 [ChatGPT 코드 해석기](https://docs.kanaries.net/ko/articles/chatgpt-code-interpreter)에 대해 자세히 알아보세요. - [GPT4All](https://docs.kanaries.net/ko/articles/gpt4all)이란?

AutoGPT: 텍스트 생성 및 완성을 위한 자율 AI 도구

AutoGPT는 LLM(Large Language Model)이 독립적으로 작업을 생각, 계획 및 실행할 수 있도록 하는 자율 AI 도구입니다. 이 도구를 사용하면 AI가 합리적으로 생각하고 의사 결정할 수 있으므로 더 이상 입력할 필요가 없습니다. 이 문서에서는 컴퓨터에 AutoGPT를 설치하고 사용하는 방법에 대해 설명합니다.

Auto-GPT를 사용하기 전에 알아야 할 사항

  • AutoGPT를 설치하고 사용하려면 강력한 컴퓨터가 필요하지 않습니다. 괜찮은 컴퓨터라면 무엇이든 할 수 있습니다. 번거로운 작업은 클라우드의 OpenAI API를 통해 이루어집니다.
  • GPT-4 API에 액세스할 수 있는 경우 생각하고 결론을 내리는 것이 훨씬 뛰어난 AutoGPT가 최적으로 작동합니다. 그러나 GPT-3.5 모델에서는 일반 OpenAI API를 사용할 수도 있습니다.
  • Auto-GPT는 대량의 토큰을 사용하므로 무료 크레딧 한도가 사라지면 결제 세부정보를 추가하고 지출 한도를 설정해야 합니다.
  • ChatGPT의 무료 버전과 달리 Auto-GPT는 인터넷에 연결하여 모든 테마에 대한 최신 정보를 찾을 수 있습니다. 따라서 이를 사용하여 모든 웹 페이지에 액세스하고 정보를 검색할 수 있습니다.

PC에서 Auto-GPT를 설정하는 방법

AutoGPT를 PC에 설치하려면 다음과 같이 하십시오.

1단계: Python 및 Pip 설치

첫 번째 단계는 공식 웹 사이트에서 Python의 최신 버전을 다운로드하여 컴퓨터에 설치하는 것입니다. Python을 설치하기 전에 Windows에서 Python.exe를 PATH에 추가를 사용하도록 설정해야 합니다. 설치 후 Windows 터미널을 열고 다음 명령을 실행하여 설치를 확인합니다. 

pip - 버전

2단계: Auto-GPT를 사용하기 위한 API 키 추가

Auto-GPT의 GitHub 릴리스 페이지로 이동하여 소스 코드(zip)를 클릭하여 ZIP 파일을 다운로드합니다. ZIP 파일의 압축을 풀고 Auto-GPT 폴더를 복사합니다. 폴더를 원하는 위치에 붙여넣습니다.

폴더를 열고 '.env.template' 파일을 찾습니다. 이 파일은 메모장이나 텍스트 편집기를 사용하여 열어야 합니다.

OpenAI 웹 사이트로 이동하여 API 키를 가져옵니다. 계정이 없는 경우 무료 계정을 만들고 새 비공개 키 만들기를 클릭합니다. API 키를 복사합니다.

텍스트 편집기로 이동하여 OPENAI_API_KEY 섹션에 API 키를 붙여넣습니다.

Pinecone.io를 열어 무료 계정을 만듭니다. 왼쪽 사이드바에서 API 키를 클릭한 다음 오른쪽 창에서 API 키 만들기를 클릭합니다.

'autogpt'와 같은 이름을 지정하고 '키 만들기'를 클릭합니다.

버튼을 클릭하여 "키 값"을 복사합니다.

텍스트 편집기를 다시 열고 PINECONE_API_KEY 옆에 붙여넣습니다. 환경 아래의 값을 복사합니다. 이것을 PINECONE_ENV 옆에 붙여 넣습니다.

".env.template" 파일을 저장합니다. .env.template의 이름을 .env로 바꿉니다.

3단계: Windows, macOS 및 Linux에 Auto-GPT 설치

Auto-GPT를 설치하려면 다음 단계를 따르세요.

  1. API 키를 추가한 후 Auto-GPT를 실행합니다. Auto-GPT 폴더를 마우스 오른쪽 단추로 클릭하고 경로로 복사를 선택합니다.
  2. 다음에 터미널을 열고 cd를 입력하고 공백을 추가한 다음 위에 복사한 경로를 붙여넣습니다. 터미널을 마우스 오른쪽 버튼으로 클릭하여 패스를 빠르게 붙여넣을 수 있습니다. 그런 다음 Enter 키를 누르면 Auto-GPT 폴더로 이동합니다. macOS 및 Linux 사용자는 폴더 자체를 마우스 오른쪽 버튼으로 클릭하여 터미널을 열면 됩니다.
  3. 여기까지 오면 다음 명령을 실행합니다. Auto-GPT를 실행하는 데 필요한 모든 라이브러리가 자동으로 설치됩니다. 
pip install -r 요구 사항.txt
  1. 그런 다음 다음 명령을 실행하면 시스템에서 Auto-GPT가 시작됩니다. 
Python -m autogpt

PC에서 Auto-GPT를 사용하는 방법

Auto-GPT를 설치한 후 사용 방법을 알아보세요. 따라야 할 절차는 다음과 같습니다.

  1. Auto-GPT는 처음 실행할 때 AI 이름을 지정하라는 메시지를 표시합니다. 예를 들어 Amazon에서 제품을 검색할 수 있는 Shopper-GPT라는 도구를 개발하고 있습니다. 특정 사용 사례에 대한 AI를 빌드하지 않으려면 이 영역을 비워두고 Enter 키를 누릅니다. 기본적으로 Entrepreneur-GPT 이름이 로드됩니다.
  2. 다음 단계는 AI의 기능을 지정하는 것입니다.
  3. 다음에 자율 AI의 일련의 목표를 설정합니다. 이제 달성하고자 하는 일을 AI에 알릴 수 있습니다. 데이터를 텍스트 또는 PDF 파일로 저장하도록 요청할 수 있습니다. 모든 정보를 얻은 후 종료를 요청할 수도 있습니다.
  4. Auto-GPT가 생각을 시작합니다. 조치가 실행되면 해당 조치를 승인하라는 메시지가 표시됩니다. 확인하려면 y를 입력하고 Enter 키를 누릅니다. 웹 사이트에 연결하여 데이터를 수집할 수 있습니다.
  5. AI의 생각, 정당화, 계획을 읽을 수 있습니다. 적절한 유형의 정보를 생성하기 위해 비판 (일종의 부정적인 제안)도 제공합니다. 그런 다음 작업이 완료됩니다.

자동 GPT 사용

Auto-GPT는 다양한 작업을 지원할 수 있는 사용하기 쉬운 자율 AI 도구입니다. 다음은 몇 가지 예입니다.

텍스트 생성

텍스트 작성은 Auto-GPT의 주요 사용 사례 중 하나입니다. 이 도구는 다음과 같은 긴 자료를 작성하는 데 사용할 수 있습니다. 제품 설명과 완전한 에세이도 포함됩니다. Auto-GPT 인터페이스의 텍스트 버튼을 클릭하고 텍스트를 작성할 프롬프트를 입력합니다. 예를 들어, 요가의 장점에 대한 기사를 만들려면 검색창에 요가의 장점을 입력하고 생성을 선택합니다.

그런 다음 프롬프트에 따라 Auto-GPT는 단락과 제목을 포함한 전체 기사를 생성합니다. 텍스트 길이, 온도(출력 예측 불가능성 결정) 및 AI가 텍스트 생성을 시도하는 횟수는 모두 변경될 수 있습니다.

텍스트 요약

Auto-GPT를 사용하면 텍스트를 빠르게 요약할 수도 있습니다. 이렇게 하려면 요약 버튼을 클릭하고 요약할 텍스트를 붙여넣습니다. Auto-GPT는 키워드와 가장 중요한 포인트가 있는 텍스트의 개요를 생성합니다.

작업 완료

Auto-GPT를 사용하여 작업을 자동으로 완료할 수도 있습니다. 예를 들어, 이 응용 프로그램을 사용하여 소셜 미디어에 게시를 계획하고 이메일 회신을 자동화할 수 있습니다. 이렇게 하려면 Auto-GPT 인터페이스의 작업 버튼을 클릭하고 자동화할 작업을 선택합니다. 예를 들어 전자 메일에 대한 답변을 자동화하려면 전자 메일을 선택하고 화면의 지시를 따릅니다.

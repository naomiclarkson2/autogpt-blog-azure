AutoGPT: テキストの生成と補完のための自律型 AI ツール

AutoGPT は、LLM (Large Language Model) が独立してアクションを考え、計画し、実行できるようにする自律型 AI ツールです。 このツールを使用すると、AI が合理的に考えて意思決定できるため、入力を行う必要がなくなりました。 この記事では、コンピューターに AutoGPT をインストールして使用する方法について説明します。

Auto-GPT を使用する前に知っておくべきこと

  • AutoGPT をインストールして使用するのに強力なコンピューターは必要ありません。 まともなコンピュータなら何でもできます。 面倒な作業はクラウド上の OpenAI API によって行われます。
  • GPT-4 API にアクセスできる場合は、考えて結論を出すのがはるかに優れている AutoGPT が最適に機能します。 ただし、GPT-3.5 モデルでは通常の OpenAI API を使用することもできます。
  • Auto-GPT は大量のトークンを使用するため、無料クレジット制限がなくなったら、請求の詳細を追加し、支出制限を設定する必要があります。
  • ChatGPT の無料版とは異なり、Auto-GPT はインターネットに接続して、あらゆるテーマに関する最新情報を見つけることができます。 したがって、これを使用して任意の Web ページにアクセスし、情報を取得できます。

PC で Auto-GPT をセットアップする方法

AutoGPT を PC にインストールするには、次の手順に従います。

ステップ 1: Python と Pip をインストールする

最初のステップは、公式 Web サイトから Python の最新バージョンをダウンロードし、コンピューターにインストールすることです。 Python をインストールする前に、Windows で「Python.exe を PATH に追加」を必ず有効にしてください。 インストール後、Windows ターミナルを開き、次のコマンドを実行してインストールを確認します。

pip --バージョン

ステップ 2: Auto-GPT を使用するための API キーを追加する

Auto-GPT の GitHub リリースページにアクセスし、「ソースコード (zip)」をクリックして ZIP ファイルをダウンロードします。 ZIP ファイルを解凍し、「Auto-GPT」フォルダーをコピーします。 フォルダーを任意の場所に貼り付けます。

フォルダーを開いて「.env.template」ファイルを見つけます。このファイルはメモ帳またはテキスト エディターを使用して開く必要があります。

OpenAI の Web サイトにアクセスして API キーを取得します。 アカウントをお持ちでない場合は、無料アカウントを作成し、「新しい秘密キーの作成」をクリックします。 API キーをコピーします。

テキストエディターに移動し、「OPENAI_API_KEY」セクションに API キーを貼り付けます。

Pinecone.io を開いて無料アカウントを作成します。 左側のサイドバーで「API キー」をクリックし、右側のペインで「API キーの作成」をクリックします。

「autogpt」などの名前を付け、「キーの作成」をクリックします。

ボタンをクリックして「キー値」をコピーします。

テキスト エディターを再度開き、PINECONE_API_KEY の横に貼り付けます。 「環境」の下の値をコピーします。 これを PINECONE_ENV の横に貼り付けます。

「.env.template」ファイルを保存します。 .env.template の名前を .env に変更します。

ステップ 3: Windows、macOS、および Linux に Auto-GPT をインストールする

Auto-GPT をインストール するには、次の手順に従います。

  1. API キーを追加したら、Auto-GPT を実行します。 「Auto-GPT」フォルダーを右クリックし、「パスとしてコピー」を選択します。
  2. 次に、ターミナルを開いて「cd」と入力し、スペースを追加して、上でコピーしたパスを貼り付けます。 ターミナルを右クリックしてパスをすばやく貼り付けることができます。 次に Enter キーを押すと、Auto-GPT フォルダーに移動します。 macOS および Linux ユーザーは、フォルダー自体を右クリックしてターミナルを開くだけです。
  3. ここまで来たら、以下のコマンドを実行します。 Auto-GPT を実行するために必要なライブラリがすべて自動的にインストールされます。
pip install -r 要件.txt
  1. 次に、以下のコマンドを実行すると、システム上で Auto-GPT が開始されます。
Python -m autogpt

PC で Auto-GPT を使用する方法

Auto-GPT をインストールしたら、その使用方法を学びましょう。従うべき手順は次のとおりです。

  1. Auto-GPT では、初回実行時に AI に名前を付けるよう求められます。 たとえば、私は Amazon で商品を検索できる Shopper-GPT というツールを開発しています。 特定のユースケース用の AI を構築したくない場合は、この領域を空のままにして Enter キーを押します。 デフォルトでは、Entrepreneur-GPT 名がロードされます。
  2. 次のステップは、AI の機能を指定することです。
  3. 次に、自律型 AI の一連の目標を設定します。 ここで、達成したいことを AI に伝えることができます。 データをテキストまたは PDF ファイルとして保存するようにリクエストできます。 すべての情報を取得した後、シャットダウンを要求することもできます。
  4. Auto-GPT が思考を開始します。 アクションが実行されると、そのアクションを承認するように求められます。 確認するには、「y」を入力して Enter を押します。 Web サイトに接続してデータを収集する可能性があります。
  5. AI の考え、正当化、計画を読み取ることができます。 適切な種類の情報を生成するために、批判 (一種の否定的な提案) も提供します。 その後、アクションが完了します。

自動 GPT の使用

Auto-GPT は、さまざまなタスクを支援できる、使いやすい自律型 AI ツールです。以下にいくつかの例を示します。

テキストの生成

テキストの作成は、Auto-GPT の主な使用例の1つです。このツールは、次のような長い資料を作成するために使用される場合があります。製品の説明や完全なエッセイも含まれます。Auto-GPT インターフェイスの「テキスト」ボタンをクリックし、テキストを作成するためのプロンプトを入力します。たとえば、ヨガの利点に関する記事を作成したい場合は、検索バーに「ヨガの利点」と入力し、「生成」を選択します。

その後、プロンプトに応じて、Auto-GPT は段落と見出しを含む記事全体を生成します。テキストの長さ、温度（出力の予測不可能性を決定する）、AI がテキストの生成を試行する回数はすべて変更される可能性があります。

テキストの要約

Auto-GPT を使用すると、テキストをすばやく要約することもできます。これを行うには、「要約」ボタンをクリックし、要約したいテキストを貼り付けます。Auto-GPT は、キーワードと最も重要なポイントを備えたテキストの概要を生成します。

タスクの完了

Auto-GPT を使用すると、タスクを自動的に完了することもできます。たとえば、このアプリケーションを使用して、ソーシャル メディアへの投稿を計画したり、電子メールの返信を自動化したりできます。これを行うには、Auto-GPT インターフェイスの「タスク」ボタンをクリックして、自動化したいジョブを選択します。たとえば、電子メールへの回答を自動化したい場合は、「電子メール」を選択し、画面上の指示に従います。
